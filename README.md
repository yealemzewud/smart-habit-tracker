# 🧠 Smart Habit Tracker

## 📌 Overview

Smart Habit Tracker is a full-stack web application that helps users build and maintain daily habits while providing meaningful insights into their behavior.


## 🚀 Features

* Add and manage daily habits
* Track habit completion (done / not done)
* View streaks (consecutive days of success)
* Identify best and worst performance days
* Simple progress visualization

---

## 🧱 Tech Stack

* Frontend: React + TypeScript
* Backend: Node.js + Express

---

## 💡 Purpose

The goal of this project is not just to track habits, but to analyze user behavior and provide insights that help improve consistency and productivity.

---

## 🎯 Key Learning Outcomes

* Full-stack development (frontend + backend)
* API communication between client and server
* State and data management
* Basic data analysis and visualization
* User experience (UX) design

---

## 🔥 Summary

This project demonstrates how simple data tracking can be transformed into meaningful insights, turning a basic habit tracker into a smart decision-making tool.
